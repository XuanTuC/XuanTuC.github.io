.NET Farmework 是什么

.NET Framework是一个用于Windows的新托管代码的编程模型。它强大功能与新技术结合起来，用于构建县有视觉上引人注目的用户体验的应用程序，实现跨技术边界的无缝通信，并且能支持各种业务流程。这些新技术有Windows Presentation Foundation、Windows CommunicationFoundation、Windows Workflow Foundation Windows Card。

托管代码(managed code)同受管制的代码，由公共语言运行库环境 (而不是直接由操作系统) 执行的代码。托管代码应用程序可以获得公共语言运行库服务，例如自动垃圾回收、运行库类型检查和安全支持等。这些服务帮助提供独立于平台和语言的、统一的托管代码应用程序行为。

公共语言运行时 (Common Language Runtime,CLR):是一套完整的、高级的虚拟机，它被设计为用来支持不同的编程语言，并支持它们之间的互操作。
