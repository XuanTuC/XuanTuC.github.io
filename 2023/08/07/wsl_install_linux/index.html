重启pc

下载 适用于 x64 计算机的 WSL2 Linux 内核更新包：

https://wslstorestorage.blob.core.windows.net/wslblob/wsl_update_x64.msi

安装更新包

下载Linux系统

到微软应用商店搜Linux，选择需要的下载

下载完毕后打开，如果成功会提示输入用户名和密码
