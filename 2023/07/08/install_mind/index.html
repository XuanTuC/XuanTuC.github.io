mindspoer 的安装
7月 8, 2023

官方教程：https://www.mindspore.cn/install

前排提示：建议先用 火绒断网修复工具 检测一下自己网络环境是否有问题，有时候开代理会影响安装

一、按照如图选择配置

Windows平台由于我也不知道的原因，只支持cpu平台，python版本根据自己的选择

二、进行SHA-256完整性校验。以2.0.0版本为例，执行以下命令：

set MS_VERSION=2.0.0

三、执行上面配置的命令：

pip install https://ms-release.obs.cn-north-4.myhuaweicloud.com/2.0.0/MindSpore/cpu/x86_64/mindspore-2.0.0-cp39-cp39-win_amd64.whl –trusted-host ms-release.obs.cn-north-4.myhuaweicloud.com -i https://pypi.tuna.tsinghua.edu.cn/simple

四、检测常见依赖是否安装：

1、点击查看需要的依赖

2、查看本地依赖版本：（以nmap为例）

pip show nmap

其中version参数即为版本号

若显示 WARNING: Package(s) not found: pack_name

则表示你的电脑上没有这个依赖，需要进行安装

3、安装需要的依赖（以nmap为例，使用清华镜像）

pip install nmap -i https://pypi.tuna.tsinghua.edu.cn/simple

对列表中的依赖每个都进行2、3步骤操作

4、单独检测****pillow依赖包的版本，确保ANTIALIAS方法可以正常使用

pip show pillow

一般显示的版本为10.0.0，因为这个版本删除了ANTIALIAS方法

因此可以选择使用9.5.0版本，或者使用 PIL库

卸载当前版本：

pip uninstall -y Pillow

安装9.5.0版本：

pip install Pillow==9.5.0 -i https://pypi.tuna.tsinghua.edu.cn/simple

参考

五、验证mindspore安装是否成功：

python -c “import mindspore;mindspore.run_check()”

如果出现

则视为安装成功，可以愉快地玩耍了!

如果报错

则为pillow包问题，参考第4步
